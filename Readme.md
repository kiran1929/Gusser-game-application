This Java program implements a simple number guessing game using object-oriented programming concepts. The game involves four roles: Guesser, Players, and an Umpire.

The gusser class asks one user (the guesser) to enter a secret number.

The player class allows three players to predict the number entered by the guesser.

The umpire class acts as the controller of the game. It:

Collects the number from the guesser.

Collects predictions from the three players.

Compares the guessed number with each player's prediction.

If a player’s prediction matches the guesser's number, that player wins the game.

If none of the predictions match, the program prints that no one won and the game should be tried again.

The gusserGameApplication class contains the main() method, which starts the game by creating an umpire object and coordinating the game flow.
